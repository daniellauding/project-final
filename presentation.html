Arkitektur

Context API

Autentisering — login, register, token-hantering. Wrappar hela appen med Provider.

Zustand

Poll-data — global store som delas mellan Home och Dashboard. Inget prop drilling.

REST API

20+ endpoints. Token-auth, anonym röstning, lösenordsskyddade polls, filuppladdning.

Zustand för server-state, Context för auth-state — varje verktyg där det passar bäst.

Hooks & Tester

Custom Hooks

useDebounce

Fördröjer uppdateringar — embed-URL laddas inte om på varje knapptryck.

useMediaQuery

Spårar viewport i JS — anpassar layout för mobil vs desktop.

usePollStore

Zustand — global poll-state med fetch, delete, reset.

28 Tester

useMediaQuery — 4 tester

Mock matchMedia, cleanup, viewport-ändring.

pollStore — 5 tester
